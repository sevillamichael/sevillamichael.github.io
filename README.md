# Online Resume

Static resume site for Michael Sevilla, hosted on GitHub Pages.

**Live:** [sevillamichael.github.io](https://sevillamichael.github.io)

## Stack

HTML, CSS, and vanilla JavaScript. No build step, no dependencies.

## Structure

```
index.html   — content and layout
styles.css   — styling and responsive design
script.js    — dark/light theme toggle with localStorage persistence
assets/      — company logos and project thumbnails
```

## Local Preview

Open `index.html` in a browser.

## Deployment

Pushes to `main` are published automatically via GitHub Pages.
